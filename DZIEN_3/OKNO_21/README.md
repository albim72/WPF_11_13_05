# 02. ICommand i własny RelayCommand

## Cel
Zastąpienie zdarzenia Click komendą, aby widok wywoływał akcję ViewModelu bez logiki w code-behind.

## Najważniejsze pojęcia
ICommand, RelayCommand, Command, Binding, INotifyPropertyChanged.

## Uruchomienie
1. Otwórz plik `WpfDay3Exercise02.csproj` w Visual Studio.
2. Uruchom projekt przez F5 lub `dotnet run` w folderze ćwiczenia.
3. Przeczytaj kod XAML i C# oraz wykonaj mini-modyfikacje z instrukcji głównej DOCX.
