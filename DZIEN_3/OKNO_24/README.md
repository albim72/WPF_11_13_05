# 05. Komunikaty między ViewModelami

## Cel
Zbudowanie prostego mechanizmu komunikacji między dwoma ViewModelami bez bezpośredniego powiązania klas.

## Najważniejsze pojęcia
Messenger, MessageBus, ViewModel, ObservableCollection, luźne powiązanie.

## Uruchomienie
1. Otwórz plik `WpfDay3Exercise05.csproj` w Visual Studio.
2. Uruchom projekt przez F5 lub `dotnet run` w folderze ćwiczenia.
3. Przeczytaj kod XAML i C# oraz wykonaj mini-modyfikacje z instrukcji głównej DOCX.
