# 06. Behavior jako Attached Property

## Cel
Wprowadzenie zachowań przez attached property: naciśnięcie Enter przenosi fokus do kolejnego pola formularza.

## Najważniejsze pojęcia
Behavior, Attached Property, Keyboard, Focus, TextBox.

## Uruchomienie
1. Otwórz plik `WpfDay3Exercise06.csproj` w Visual Studio.
2. Uruchom projekt przez F5 lub `dotnet run` w folderze ćwiczenia.
3. Przeczytaj kod XAML i C# oraz wykonaj mini-modyfikacje z instrukcji głównej DOCX.
