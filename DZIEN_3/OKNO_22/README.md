# 03. CanExecute i walidacja przycisku

## Cel
Pokazanie, że komenda może sama decydować, czy przycisk ma być aktywny, na podstawie danych z formularza.

## Najważniejsze pojęcia
CanExecute, RaiseCanExecuteChanged, walidacja, TextBox, ObservableCollection.

## Uruchomienie
1. Otwórz plik `WpfDay3Exercise03.csproj` w Visual Studio.
2. Uruchom projekt przez F5 lub `dotnet run` w folderze ćwiczenia.
3. Przeczytaj kod XAML i C# oraz wykonaj mini-modyfikacje z instrukcji głównej DOCX.
