# 01. Struktura MVVM: Model, View, ViewModel

## Cel
Zrozumienie podziału aplikacji WPF na Model, View i ViewModel oraz ustawienia DataContext dla widoku.

## Najważniejsze pojęcia
MVVM, Model, View, ViewModel, DataContext, Binding.

## Uruchomienie
1. Otwórz plik `WpfDay3Exercise01.csproj` w Visual Studio.
2. Uruchom projekt przez F5 lub `dotnet run` w folderze ćwiczenia.
3. Przeczytaj kod XAML i C# oraz wykonaj mini-modyfikacje z instrukcji głównej DOCX.
