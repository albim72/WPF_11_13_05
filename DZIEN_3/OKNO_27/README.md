# 07. MVVM Toolkit: ObservableObject

## Cel
Użycie CommunityToolkit.Mvvm do skrócenia kodu ViewModelu i automatycznego generowania właściwości.

## Najważniejsze pojęcia
CommunityToolkit.Mvvm, ObservableObject, ObservableProperty, NotifyPropertyChangedFor.

## Uruchomienie
1. Otwórz plik `WpfDay3Exercise07.csproj` w Visual Studio.
2. Uruchom projekt przez F5 lub `dotnet run` w folderze ćwiczenia.
3. Przeczytaj kod XAML i C# oraz wykonaj mini-modyfikacje z instrukcji głównej DOCX.

Wymaga pakietu NuGet `CommunityToolkit.Mvvm`. Visual Studio pobierze go przy restore.
