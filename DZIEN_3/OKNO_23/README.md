# 04. CommandParameter i SelectedItem

## Cel
Użycie CommandParameter oraz SelectedItem do wykonania jednej komendy z różnymi parametrami statusu.

## Najważniejsze pojęcia
CommandParameter, SelectedItem, ICommand, status, Binding TwoWay.

## Uruchomienie
1. Otwórz plik `WpfDay3Exercise04.csproj` w Visual Studio.
2. Uruchom projekt przez F5 lub `dotnet run` w folderze ćwiczenia.
3. Przeczytaj kod XAML i C# oraz wykonaj mini-modyfikacje z instrukcji głównej DOCX.
